Core API

Model Initialization

from llama_cpp import Llama

model = Llama(
    model_path="path/to/model.gguf",
    n_ctx=2048,      # Context window
    n_threads=4,     # CPU threads
    n_gpu_layers=0   # GPU layers
)
Parameter Type Description
model_path str Path to GGUF model file
n_ctx int Context window size
n_threads int Number of CPU threads

Model Loading

Loading Methods

# Standard loading
model = Llama.from_pretrained(
    "model_name",
    gpu_layers=0
)

# Quantized loading
model = Llama.from_quantized(
    "model_path",
    quantization="q4_k_m"
)

Loading Tips

  • Use appropriate quantization for your hardware
  • Monitor memory usage during loading
  • Consider lazy loading for large models

Inference

Text Generation

# Basic generation
output = model.generate(
    "Your prompt here",
    max_tokens=100,
    temperature=0.7,
    top_p=0.95
)

# Streaming generation
for token in model.generate_stream(
    "Your prompt here",
    max_tokens=100
):
    print(token, end="", flush=True)

Important Notes

  • Monitor token count to avoid context overflow
  • Handle stream interruptions gracefully
  • Consider batch processing for multiple requests

Parameters

Generation Parameters

Parameter Range Description
temperature 0.0 - 2.0 Controls randomness
top_p 0.0 - 1.0 Nucleus sampling
top_k 1 - 100 Top-K sampling 
# Example with all parameters
output = model.generate(
    prompt="Hello",
    max_tokens=100,
    temperature=0.7,
    top_p=0.95,
    top_k=40,
    repeat_penalty=1.1,
    presence_penalty=0.0,
    frequency_penalty=0.0
)

Utilities

Helper Functions

# Token counting
n_tokens = model.token_count("Your text here")

# Context management
model.reset_context()
model.clear_cache()

# Model info
print(model.model_info())
print(model.get_memory_usage())

Best Practices

  • Regular cache clearing
  • Memory monitoring
  • Context reuse when possible
